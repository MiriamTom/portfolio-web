Bahcelor Thesis - Health Receiver

Health Receiver is a web application used to display users' health and location data obtained from the cloud. Data is collected from wearable devices (e.g. Huawei watches) and sent via the MQTT protocol.

Languages:

JavaScript, MQTT, HTML, CSS, WebSockets

Purpose:

Pre efektívne zobrazovanie a spracovanie údajov zo zdravotných wearables v reálnom čase.

Outcome::

Zabezpečená robustná vizualizácia zdravotných a lokalizačných dát pre lepšiu kontrolu používateľa nad svojím zdravím.

RATMAZE game

Ratmaze is a JavaFX-based maze game where players control a rat through intricate mazes, collect cheese, avoid cats, and reach the exit point to progress through levels. The game features background music and offers an immersive experience with dynamic gameplay elements.

Languages:

Java, JavaFX

Purpose:

Create a fun and interactive game to improve programming skills in JavaFX.

Outcome::

A complete game with multiple levels that has received positive feedback from users.

2048

This project contains the game 2048 implemented in the C language. 2048 is a single-player sliding tile puzzle game where the goal is to combine tiles to reach the magic number: 2048.

Two years after creating the game, an AI model was added — because why play yourself when your code can play for you (and probably better)?

Languages:

C, Python

Purpose:

To learn and implement basic algorithms and data structures in C.

Outcome:

Fully playable version of 2048, later enhanced with an AI in Python that can solve the game — or at least pretend to be a genius.
